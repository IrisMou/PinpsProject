# 顶盛体育-优选体育平台

  该APP是一款服务于爱好乒乓球运动的应用。
  主要是提供线上购买乒乓球用品(乒乓球装备、乒乓球书籍)等乒乓球商品；
  特地是给爱好乒乓球的人，提供一个购买乒乓球商品的平台。
    
    首页：乒乓球装备、乒乓球书籍、国球必备、新品上市、单品优选和大家都在买等。
    
    积分商城：用户购买商品获得对应的积分，和签到获取积分，可直接在"积分商城"兑换商品。
    
    订单(全部/已下单/运送中/已收货/可售后订单)等。
